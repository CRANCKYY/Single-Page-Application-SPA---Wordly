# 🎬 CRANCKY flix - Movie Collection SPA

## 📽️ Project Overview
CRANCKY flix is a **Single Page Application (SPA)** that allows users to manage their personal movie collection. Users can add, view, search, filter, and delete movies, save favorites using localStorage, and watch YouTube trailers for their movies.

## 🎯 Problem Solved
Movie enthusiasts often struggle to remember which movies they've watched, their ratings, and thoughts about films. CRANCKY flix solves this by providing a simple, elegant interface to:
- Keep a personal record of watched movies
- Rate and review films
- Search and filter through collections
- Save favorite movies for quick access
- Watch movie trailers instantly

## 🛠️ Technologies Used
- HTML5 (Semantic markup)
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- localStorage for data persistence
- YouTube Data API v3
- Google Fonts (Inter)

## ✨ Features
- **Single Page Application** - No page refresh
- **4 Views**: Home, Collection, Favorites, Add Movie
- **Responsive Design** - Works on mobile, tablet, and desktop
- **Form Validation** - Real-time validation with error messages
- **localStorage** - Data persists between sessions
- **Search & Filter** - Find movies by title or genre
- **Favorites** - Save and manage favorite movies with ❤️
- **Visual Saved State** - Highlighted favorites with ⭐ and gold border
- **YouTube Trailers** - Watch official movie trailers with one click

## 📱 Responsive Breakpoints
- Desktop: >1024px
- Tablet: 768px - 1024px
- Mobile: <768px

## 🚀 How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Navigate to "Add Movie" to add your first movie
4. Fill out the form (title, genre, year, rating, review)
5. View all movies in "Collection"
6. ❤️ Click the heart to save favorites
7. View favorites in the "Favorites" tab
8. ▶️ Click "Trailer" to watch the official movie trailer
